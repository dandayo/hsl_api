# HSL API — Helsinki Public Transport App

A simple React app that helps you navigate Helsinki’s public transport. Check live bus schedules, plan routes, see current weather, and search stops or places quickly.

## Features
- **Real-time Bus Schedules**
- **Route Planning** between any two locations
- **Weather Info** for Helsinki
- **Search Stops & Places** with autocomplete

## Tech
- React 19
- Vite
- React Router
- Vercel
- HSL Digitransit API
- OpenWeatherMap API

## What I Learned
- Working with external APIs (HSL, OpenWeatherMap)
- Serverless functions & proxy setup
- React Router navigation
- GraphQL queries for routes
- Debouncing search input
